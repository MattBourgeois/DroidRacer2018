#How to have pi autostart VNC Server on boot for easy connection without peripherals.

#Current authentication details:
For network "bootsandpants":
	IP: 192.168.43.7
	Password: droidracer


1. cd home/pi/.config/autostart/

2. nano tightvnc.desktop

3. Type the following:
	[Desktop Entry]
	Type=Application
	Name=TightVNC
	Exec=vncserver :1
	StartupNotify=false

4. Hit "CTRL+X" and then "Y"

Done.
